## 部署

纯静态页面，直接部署到 GitHub Pages：

1. 创建 GitHub 仓库 `energy-toolbox`
2. Settings → Pages → Source: `main` branch, root directory
3. 访问 `https://<username>.github.io/energy-toolbox`
